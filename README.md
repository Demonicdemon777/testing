# AI Resume Analyzer

An AI-powered web application that analyzes resumes, provides improvement suggestions, generates interview questions, and evaluates answers to help candidates prepare for technical interviews.

## 🚀 Features

* **Resume Upload**

  * Upload resumes in PDF format
  * Automatic text extraction from the resume

* **AI Resume Analysis**

  * Resume quality score
  * Suggestions to improve resume content
  * Grammar and clarity improvements
  * LinkedIn summary generation

* **ATS Optimization**

  * Evaluates resume compatibility with ATS systems
  * Highlights missing skills

* **Interview Preparation**

  * Generates technical interview questions
  * Evaluates user answers
  * Provides feedback and improvement suggestions

* **Modern UI**

  * Clean dashboard with resume score
  * Responsive interface built with React

---

## 🛠️ Tech Stack

### Frontend

* React
* Axios
* CSS

### Backend

* Node.js
* Express.js
* Multer (file uploads)
* pdf-parse (resume text extraction)

### AI Integration

* Groq API (LLM for resume analysis and interview questions)

## ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/ai-resume-analyzer.git
```

### 2️⃣ Install backend dependencies

```
cd backend
npm install
```

### 3️⃣ Install frontend dependencies

```
cd ..
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file in the backend folder:

```
PORT=5000
GROQ_API_KEY=your_groq_api_key_here
```

---

## ▶️ Running the Application

### Start Backend

```
cd backend
npm start
```

### Start Frontend

```
npm run dev
```

The application will run at:

```
http://localhost:5173
```

---

## 📊 How It Works

1. User uploads a resume (PDF)
2. Backend extracts resume text
3. AI analyzes resume content
4. System generates:

   * Resume score
   * Improvement suggestions
   * ATS feedback
5. User can also practice interview questions generated by AI
